# Automations

A collection of automation workflows built with n8n and AI agents.

## Projects

### 1) AI Work Log Assistant (Telegram + n8n + Google Sheets)
A conversational assistant that logs work hours from natural language messages (Spanish/English), requires confirmation, stores pending entries, and writes confirmed logs to Google Sheets.

📁 Project folder: `ai-worklog-assistant/`

Key docs:
- Architecture: `ai-worklog-assistant/docs/architecture.md`
- Prompt: `ai-worklog-assistant/prompt/system-prompt.txt`
- Examples: `ai-worklog-assistant/examples/sample-messages.md`
- n8n export: `ai-worklog-assistant/workflow.json`
